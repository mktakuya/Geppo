# Geppo
'Geppo' is a simple and fast generating monthly reports for Hatena Blog.

## Installation

```
git clone https://github.com/mktakuya/Geppo.git
cd Geppo
bundle install
```

## Usage

```
ruby geppo.rb -y 2015 -m 8
```
